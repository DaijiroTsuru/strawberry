## 主な登録タスク（概要）

### フェーズ1: 準備（1週間）
- 本人確認書類、銀行口座情報の準備
- 全11商品の情報整理と価格確認
- 商品写真撮影（各商品5枚以上、農園風景10枚以上）
- 農園紹介文と各商品説明文の作成
- 梱包資材の在庫確認

### フェーズ2: アカウント登録（2〜3日）
- 食べチョク生産者アカウント作成
- 農園プロフィール・紹介文の登録
- 農園画像のアップロード
- 銀行口座情報の登録

### フェーズ3: 商品登録（1週間）
- いちご商品4品の登録
- お米商品5品の登録
- もち商品2品の登録
- 各商品の価格、説明文、画像、配送設定

### フェーズ4: テスト・運用開始（3日）
- テスト注文の実施
- プロモーション準備
- カスタマーサポート体制構築
- 正式オープン

## 重要なポイント

### 津留いちご園の強み
1. **かおり野の土耕栽培**: 豊かな香りと味わい
2. **有機質肥料の使用**: 化学肥料に頼らない栽培（油粕、米糠、海藻、魚粉、黒砂糖など）
3. **無農薬米**: 除草剤・農薬不使用、12種類の有機質肥料使用
4. **地域性**: 福岡県筑後市、八女ICより20分

### 食べチョクのメリット
1. **直接販売**: 中間業者を通さず消費者に直接販売
2. **高い利益率**: 市場価格より高く販売可能
3. **大規模集客**: 月間100万人以上のユーザー
4. **ブランディング**: ストーリーや栽培方法をアピール可能

### Shopify併用の戦略
- 在庫管理の一元化が必要
- 価格設定の統一または明確な差別化
- 食べチョクは「お試しセット」「季節限定品」に特化する選択肢も

## 次のステップ

1. **本ガイドの確認**: `guides/TABECHOKU_REGISTRATION.md` を熟読
2. **準備開始**: チェックリスト（`guides/TABECHOKU_TASKS_CHECKLIST.md`）に沿って準備開始
3. **写真撮影**: 商品写真と農園風景の撮影（最重要）
4. **文章作成**: 農園紹介文と商品説明文の作成
5. **登録実施**: 食べチョク生産者登録ページから登録開始

## 推定作業時間

- **初回準備・登録**: 2〜3週間
- **日常運用**: 1日30分〜1時間（注文確認、在庫更新、問い合わせ対応）
- **定期メンテナンス**: 週1時間、月2時間
