栽培方法へのこだわり

津留いちご園は、手間とコストをかけても、皆様に安全でおいしいお米をご提供したいと願い、丁寧にお米の栽培を行っています。 大事に思っているのは「いかに無農薬で強く美味しいお米をつける稲を育てるか」。津留いちご園は下記の栽培方法で、おいしいお米を追求しています。

  一般の農園 津留いちご園
種まき（※1） 一箱あたり200g 一箱あたり80gの薄まき
代かき（※2） 2回 3回
本田植え付け（※3） 70～80株 45株
除草剤 除草剤を使用する 除草剤を使用しない
肥料 化学肥料のみ 油粕、米糠、ケイフン、海草、
大豆粕、魚粉、堆肥その他を
発酵させ使用
農薬散布 農薬散布を行う 散布は一切行わない
  • ※1 種まきを薄くすると補植（田植え機が植えそこなった所を人の手で植えていく作業）が大変ですが、しっかりとした苗になり病害虫が減少します。
  • ※2 代かき（田植えのために、田に水を入れて土を砕いてかきならす作業）の回数を増やすと、手間が掛かりますが、雑草の数が減少します。
  • ※3 本田植え付け本数を減らすと、収穫量は減少しますが、風通しが良く、しっかりとした草体となり、病害虫に強くなります。
  • ※4 有機肥料を施用すると、肥料代や散布のコストが多くかかりますが、ビタミン・ミネラル・アミノ酸の種類と量が豊富になり、味がよくなります。
